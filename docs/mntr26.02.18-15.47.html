Installation

git clone https://github.com/nafiurohman/system-monitoring.git
cd system-monitoring
pip3 install -r requirements.txt
python3 app.py

Access Dashboard

Buka browser: http://127.0.0.1:9999

Fitur Utama

  • CPU Monitoring: Per-thread usage, temperature, load average
  • GPU Monitoring: Multi-vendor (NVIDIA/AMD/Intel), cards + charts
  • Memory: Detailed breakdown dengan ZRAM support
  • Network: 4 status indicators, traffic details
  • Security: Score, firewall, antivirus, ports analysis
  • Process Manager: Task Manager style dengan search/sort
  • Storage Browser: Dropdown dengan large files detection

Dashboard Pages

  1. Overview - System info, CPU, memory, disk, network dengan charts
  2. Server Status - Health monitoring, services, ports, alerts
  3. Hardware - Performance metrics, thermal, CPU, GPU, memory, disk
  4. Network - Status indicators, interfaces, traffic, firewall
  5. Services - System services monitoring
  6. Processes - Process Manager dengan search/sort
  7. Security - Security score, firewall, antivirus, ports
  8. Storage - Storage browser dengan dropdown
  9. Docker - Container dan image monitoring
  10. Kubernetes - Pods, services, deployments
  11. Logs - System dan auth logs dengan JSON export
