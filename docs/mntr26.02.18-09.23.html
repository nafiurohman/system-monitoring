Tentang System Monitor

System Monitor adalah platform monitoring server profesional yang dibangun dengan Python + Flask untuk performa optimal dan monitoring real-time. Aplikasi ini menyediakan monitoring lengkap untuk CPU, GPU, Memory, Network, Security, dan banyak lagi.

Fitur Utama

1. CPU Monitoring

  • Per-Thread Monitoring: Monitoring usage setiap thread CPU secara real-time
  • CPU Information: Model, cores, threads, frequency, cache size
  • Temperature Monitoring: Monitoring suhu CPU dengan alert system
  • Load Average: 1 minute, 5 minutes, 15 minutes
  • Context Switches: Monitoring context switch rate
  • Interrupt Rate: Monitoring interrupt per detik

2. GPU Monitoring

  • Multi-Vendor Support: NVIDIA, AMD, Intel
  • GPU Cards: Card terpisah untuk setiap GPU yang terdeteksi
  • Dual Chart: Chart utilization dan temperature dalam satu tampilan
  • Memory Monitoring: Total, used, free GPU memory
  • Temperature: Real-time GPU temperature dengan color-coding
  • Utilization: Persentase penggunaan GPU

3. Memory Monitoring

  • Detailed Breakdown: Used, free, available, buffers, cached, shared
  • Active/Inactive Memory: Monitoring memory yang aktif dan tidak aktif
  • ZRAM Support: Detection dan monitoring ZRAM (Linux)
  • Swap Monitoring: Total, used, free, sin, sout
  • Visual Progress Bar: Color-coded (green/yellow/red)

4. Network Monitoring

  • 4 Status Indicators:
    • Monitor Status (Active/Inactive)
    • Internet Status (Connected/Disconnected via ping 8.8.8.8)
    • Device Status (Online/Offline)
    • Active Connections (Count)
  • Per-Interface Details: Upload/download rate real-time
  • Packet Rate: Packets sent/received per second
  • Error & Drops: Detection error dan packet drops
  • Firewall Status: UFW, iptables, Windows Defender

5. Performance Deep Metrics

  • Context Switches: Total context switch count
  • Interrupts: Total interrupt count
  • File Descriptors: Current usage vs limit
  • Thread Count: Total threads di sistem
  • Zombie Processes: Detection zombie process
  • Resource Limits: Max open files, processes, sockets

6. Thermal & Power Monitoring

  • CPU Temperature: Current, average, status (Normal/Warning/Critical)
  • Battery Status: Percent, plugged status, time remaining
  • Thermal Throttling: Detection throttling events
  • Power Adapter: Detection power adapter status

7. Security Monitoring

  • Security Score: 0-100 dengan label (Critical/Warning/Good/Excellent)
  • Firewall Detection: UFW, iptables, Windows Defender
  • Antivirus Status: ClamAV, Windows Defender
  • Open Ports Analysis:
    • Port number, Protocol (TCP/UDP)
    • Service name, Process name
    • Risk level (High/Medium/Low)
    • Color-coded badges
  • Failed Login Attempts: Last 20 attempts
  • Logged Users: Current users dengan timestamp
  • Suspicious Processes: Detection process berbahaya

8. Process Manager

  • Task Manager Style: Interface seperti Windows Task Manager
  • Sortable Columns: PID, Name, User, CPU%, Memory%, Threads, Status
  • Search/Filter: Real-time search functionality
  • Color-Coded: Red (>50% CPU), Yellow (>25% CPU)
  • Manual Refresh: Button refresh dengan visual feedback
  • Performance Optimized: Limit 100 processes

9. Storage Browser

  • Dropdown Navigation: Per partition dengan expand/collapse
  • Large Files Detection: Files >100MB
  • Directory Tree: Navigasi directory structure
  • File Size: Human readable format
  • SMART Status: Disk health monitoring

10. Services Monitoring

  • Windows Support: W3SVC, MSSQLSERVER, MySQL, PostgreSQL, Redis, Docker
  • Linux Support: nginx, apache2, mysql, mariadb, postgresql, redis, ssh, docker
  • Empty State: Informative message jika tidak ada service
  • Service Summary: Total dan running count

Teknologi

  • Backend: Python 3.8+ dengan Flask
  • System Library: psutil (cross-platform)
  • Frontend: HTML5 + CSS3 + Vanilla JavaScript
  • Charts: Chart.js 4.4.0
  • UI Design: Dark Blue Glass Morphism
  • Icons: Font Awesome 6.5.1

Performance

  • Auto-Refresh: 3 seconds (anti-panas)
  • Tab Visibility Detection: Pause saat tab hidden
  • Cache TTL: 0.5 second untuk ultra real-time
  • Lazy Loading: Per page loading
  • Chart Optimization: Update mode 'none'
